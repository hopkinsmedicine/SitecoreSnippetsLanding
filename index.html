Johns Hopkins Medicine Generic HTML Snippets for Sitecore CMS are a set of reusable HTML components that can be used to build web pages. These snippets are designed to be used in conjunction with the Sitecore CMS. The snippets are built using HTML, CSS, and JavaScript. Just select a snippet from the list below and copy the code into your Sitecore page. Reach out for assistance if you need any help at webcenter@jh.edu .
